Misc总结之图片隐写

发表于2023-11-28|更新于2021-02-10
|阅读量:

MISC简介

Misc为英文miscellaneous的前四个字母，是杂项、混合体、大杂烩的意思

主要类型有

  • 数据编码、图形编码
  • 图片隐写
  • 音频、视频隐写
  • 流量分析
  • 内存取证
  • 游戏隐写

图片隐写分类

右击属性

属性————详细信息 中可能会隐藏一些字符串信息

文件中藏字符串

  • strings可直接查看
  • grep -a “ctf” grep.jpeg
  • file识别文件类型

文件包含

  • binwalk -e file 分离文件
  • foremost file 提取文件
  • dd if=file of=outfile bs=1 skip=开始分离的数字位置

修改文件头

  • JPEG (jpg)，
    文件头：FFD8FF
    文件尾：FF D9
  • PNG (png)，
    文件头：89504E47
    文件尾：AE 42 60 82
  • GIF (gif)， 　　
    文件头：47494638 或GIF89A　　　　　　　　　　　　　　　　　　　
    文件尾：00 3B
  • ZIP Archive (zip)，
    文件头：504B0304　　　　　　　　　　　　　　　　　　　　　　
    文件尾：50 4B

// 以下不太常见，但也要知道呀

  • HTML (html)，
    文件头：68746D6C3E
  • Wave (wav)，
    文件头：57415645
  • AVI (avi)，
    文件头：41564920
  • bmp， 　
    文件头：424D
  • Email [thorough only] (eml)，
    文件头：44656C69766572792D646174653A
  • Adobe Acrobat (pdf)，
    文件头：255044462D312E
  • RAR Archive (rar)，
    文件头：52617221
  • Photoshop (psd)，
    文件头：38425053
  • XML (xml)，
    文件头：3C3F786D6C

GIF

  • 特殊帧（ps/stegsolve）
  • 帧的时间间隔（看出0101010类型编码）

png（bmp）

  • zilb(010/pngcheck)

  • IHDR、IDAT

  • lsb隐写/lsb加密（zsteg/stegsolve）

  • wbs43open(bmp)

jpg

  • jphide(jphs)

  • steghide

  • stegdetect

  • outguess

  • F5-steganography

  • 水印(单图)(工具+Java盲水印)
